Help - FAQ

Tab creation delay is not working:

For tab creation delay to work correctly, the Bulk URL Opener window must be left open until all tabs have been created, this is due to how javascript runs in chrome extensions

What is an experimental feature:

Many new features have been added to the extension that can not be full tested before release. So these features are marked as experimental for a period while any usability issues or bugs can be identified. They have been tested to the same extend as other features prior to release but issue may persist due to the nature of these features. If you find any issues with these features please report it to the support form here

How do I edit/delete a list when automatically open lists is enabled:

If you wish to edit or delete a list when you have automatically open lists enabled you can check the checkbox on the popup for the extension with the label 'Override Auto Open'. This will temporarily disable the auto open setting allowing you to edit and delete saved lists.
